# Telegram AI Assistant Bot

Цей бот використовує **Faster-Whisper** для розпізнавання голосу та **Llama 3.1 (Groq)** для відповідей. Також інтегровано генерацію зображень через Pollinations AI.

## Як протестувати
Ви можете спробувати бота в дії за посиланням: 
👉 [https://t.me/Av373Bot](https://t.me/Av373Bot)

## Команди
* `/start` — запуск та привітання.
* `/imagine [текст]` — генерація зображення з вибором стилю.
* `/draw [текст]` — швидка генерація зображення.
* Надішліть **голос**, щоб бот його розпізнав і відповів.
